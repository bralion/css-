目标：设置一个容器 只在内部有圆角而border或者outline依然保持直角

实验框： 
           .old-way {
                width: 220px;
                height: 220px;
                margin-left: 20px;
                margin-top: 20px;
                background:#661;
                padding-top: 10px;
                padding-left: 10px;
            }
            .old-way>div{
                border-radius:0.8em;
                width: 200px;
                height: 200px;
                background:tan;
            }
        

        background:tan;
        border-radius:0.8em;
        padding: 1em;
        box-shadow: 0 0 0 0.5em red;
        outline: .6em solid #661;
将box-shadow的颜色变成#661 则完成了目标 这里为了区别出来
注意：1.上图中利用了outline不会和border一起变成圆角，然后box-shadow生成在outline之下 来完成目标
结论和扩展：常见的实现方式是使用两个div实现，但是这样会两个样式相互关联改动起来不方便，代码量也比较大
