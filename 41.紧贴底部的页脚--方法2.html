目标：实现紧贴底部的页脚--方法1

有时候会遇到需要页脚紧贴底部的情况，并且在各个高度的情况下依然成功。之前出现过必须F11，才能正常显示的问题。

实验框： 
            代码如下：
                body{
                    display: flex;
                    flex-flow: column;
                    min-height: 100vh;
                }
                header{
                    height:70px;
                }
                main{
                    flex:1;
                }
                footer{
                    background: black;
                    color: #ffffff;
                    padding:1em;
                    height:80px;
                }

结论：采用弹性布局， flex-flow: column; 并且设置main的flex为1,使它变成可伸缩即可。此种方案，在任何高度下都能实现页脚沉于底部
