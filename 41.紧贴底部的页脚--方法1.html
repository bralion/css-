目标：实现紧贴底部的页脚--方法1

有时候会遇到需要页脚紧贴底部的情况，并且在各个高度的情况下依然成功。之前出现过必须F11，才能正常显示的问题。

实验框： 
            代码如下：
                body{
                    min-height:100vh;
                }
                header{
                    height:70px;
                }
                main{
                    min-height:calc( 100vh - 150px)
                }
                footer{
                    background: black;
                    color: #ffffff;
                    padding:1em;
                    height:80px;
                }

结论：采用vh进行如上配置后，在任何高度的页面可以实现footer一直在底部，main部分可以伸缩.
