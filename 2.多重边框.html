目标：实现目标元素的多重边框效果

实验框： 
        border-width:10px;
        border-style: dashed;
        border-image: url("./static/sucai1.jpg") 14 14 14 14 round;
border-image实现--这里的图片内容不是多层的效果
注意：1.border-color和border-image会相互覆盖 
            
            
            box-shadow: 0 0 0 10px red,0 0 0 20px green,0 0 0 30px blue;
注意：1.shadow是层层叠加的 可以看到 宽度10px 20px 30px 
                background-color:white;
                border: 10px solid darkgreen;
                outline: 5px solid blue;
outline方案只适用于两层边框 outline可以设置虚线 也可以通过 outline-offset设置偏移量
结论：
    1.border-image语法：border-image: source slice width outset repeat|initial|inherit;
    border-image-source	用于指定要用于绘制边框的图像的地址
    border-image-slice	图像边界向内偏移
    border-image-width	图像边界的宽度
    border-image-outset	用于指定在边框外部绘制 border-image-area 的量
    border-image-repeat	用于设置图像边界是否应重复（repeat）、拉伸（stretch）或铺满（round）
